Правила Perfect Present Continuous

Для времени Present Perfect Continuous правила гласят следующее: действие началось в прошлом, и все еще продолжается. Также употребляется тогда, когда требуется подчеркнуть промежуток времени, например, вы что-то делаете 3 года. Дословно оно переводится как «Настоящее совершенное длительное».

Рассмотрим примеры предложений Present Perfect Continuous:

1. I have been thinking about you. Я все еще думаю о тебе.

2. He has been working all day. Он работает весь день.

3. It has been raining for 3 hours. Дождь идет 3 часа.

А теперь распишем правила для разных типов предложений.

Утвердительное:

I/we/you/they + have + been + Ving. I have been working for 3 hours. Я работаю три часа.

She/it/he + has + been + Ving. She has been reading the book since morning. Она читает эту книгу с утра.

Вопросительное:

Have + I/we/you/they + been + Ving. Have they been working since morning? Они работают с утра?

Has + she/it/he + been + Ving. Has he been swimming for an hour? Он плавает один час?

Отрицательное:

I/we/you/they + have + not + been + Ving. I have not been working for 3 hours. Я не работаю три часа.

She/it/he + has + not + been + Ving. She has not been reading the book since morning. Она не читает эту книгу с утра.
